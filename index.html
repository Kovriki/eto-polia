  • → Совместно с владельцем продукта и бизнес-аналитиком внедряла систему Канбан для организации процесса во время роста команды. Данная настройка внутренних процессов сократила время lead time с >100 дней до ~25.

  • → Для отдельных спецпроектов разрабатывала дизайн функционала тестов, калькуляторов, ии-генераторов.
  • → Разработала дизайн рекламных форматов, расписала гайды для команды редакции по использованию новых форматов контента.
